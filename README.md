# AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes against job descriptions using NLP techniques like TF-IDF Vectorization and Cosine Similarity. The system generates ATS scores, detects missing keywords, and analyzes resume relevance for job applications.

---

# Features

- Upload Resume PDF
- Extract Resume Text
- ATS Score Calculation
- Job Description Matching
- Missing Keywords Detection
- NLP-Based Similarity Analysis
- React Frontend
- Flask Backend API
- Fast Resume Evaluation
